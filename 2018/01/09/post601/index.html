好久不写博客了

Jan 9, 2018

真的是好久好久不写博客了，主要是懒。。之前一直使用wordpress，可是国内好难找到好用的wordpress空间。之前一度以为daocloud.io是我的福音，我曾好长一段时间把博客放在daocloud上，可好用了，我也是那时候体会到了docker的强大。后来daocloud果然被监管了，我的博客被停了，于是这一年多来我的域名就被打入冷宫了。。当时数据都没来得及备份，真是遗憾。

github page是个好东西，其实以前也用过，觉得好简单，但是现在确实是过了那个爱折腾的年龄了，，我到哪去找又便宜又好用的php空间呢，于是又搭起了hexo，hexo明显比两年前好了不少，插件很丰富，配合hexo-admin书写体验也蛮好的。

今天从各种途径找到了以前的文章，还好当时用windows live写博客，本地有记录，真是感慨万千。我把以前写的大部分文章都翻出来了，迁移到hexo上了，人工迁移的，我一篇一篇阅读以前的文章，虽然不多，但看得真是百感交集，当年有些想法好搞笑，也有些文章，看得我自己都感动了。。总的来说还是蛮开心的，我发现我还是以前那个我，爱吐槽爱思考的我！

顺便发现coding.net也有pages托管，比github pages牛的，它还支持动态php页面，也就是说可以在coding.net上托管wordpress的，再绑个域名，真是完美的php免费空间！但现在真是有心无力，不想再折腾了，而且我也很担心，coding.net用得人多了，万一他也被监管了乍办。。人家做企业也不容易呀，他好意我心领了，不能给人家找麻烦。

写博客是个好习惯，时常把以前写的文章翻出来看看，可以回顾自己的生活，是非常好的体验。好习惯，贵在持之以恒！
